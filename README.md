# Personality Test Web App

Next.js + TypeScript + Tailwind CSS frontend scaffold for a personality test app.

## Pages

- `/` Home page
- `/test` Test page
- `/result` Result page

## Development

1. Install Node.js 18+ (or latest LTS)
2. Install dependencies:

```bash
npm install
```

3. Run dev server:

```bash
npm run dev
```
